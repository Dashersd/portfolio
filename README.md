# 🌌 Cyberpunk Glassmorphic Portfolio

Welcome to the official repository of my personal portfolio—a modern, high-performance, and futuristic web experience. Crafted with premium design aesthetics, rich micro-animations, and fluid transitions, this portfolio represents my journey, skills, and projects as a developer.

## ✨ Key Features & Design Details

- **🎨 Futuristic Glassmorphism**: Tailored neon and dark HSL color palettes with subtle gradient glows and dynamic blur backdrops.
- **✨ Custom Glow Cursor**: Seamlessly follows user movements with an interactive trailing cursor dot and responsive scaling effects when hovering over buttons, cards, and links.
- **🕸️ Canvas Particle Network**: An interactive, lightweight HTML5 Canvas background rendering animated node particles that dynamically connect when they are close.
- **🖼️ 3D Perspective Tilt**: Project cards respond to mouse coordinates, rendering rich 3D hover movements.
- **⚡ Intersection Observer Scroll Reveal**: Elements smoothly fade and slide into view as the user scrolls, creating a polished, narrative-driven experience.
- **🎭 Dual-Image Hover Reveal**: Profile pictures dynamically cycle through alternate poses on hover, injecting playfulness and interactivity.
- **✍️ Typing Animation Effect**: An automated, pure JavaScript-driven typewriter effect highlighting core professional focuses in real-time.
- **📱 Fluid Responsiveness**: Hand-crafted CSS grid and flex layouts adapting beautifully across standard, tablet, and mobile screens.

---

## 🛠️ The Tech Stack

This project is built using a clean, pure web development pipeline with no heavy frameworks or overhead dependencies, ensuring lightning-fast load times and maximum performance.

- **HTML5**: Structured with semantic tags and high-level readability for search engine optimization (SEO).
- **CSS3 (Vanilla)**: Utilizes advanced variables (CSS custom properties), HSL color spaces, flexbox, grid, neon text/box-shadows, and smooth hardware-accelerated transitions.
- **JavaScript (ES6+)**: Powers custom particle systems, custom cursors, forms validation, the typing animation, scroll spy highlights, and intersection observers.
- **Fonts & Icons**:
  - Google Fonts (Poppins & Montserrat)
  - Font Awesome Icons v6

## 🚀 How to Run Locally

Since this application is built on top of standard core web technologies, you don't need any complex installation processes.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dashersd/portfolio.git
   cd portfolio
   ```

2. **Open in Browser:**
   - Double-click `portfolio.html` to open it in your default web browser, OR
   - Run it with a local development server like **Live Server** in VS Code for live editing.
